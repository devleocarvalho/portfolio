O mnemônico FAST HUG foi proposto há anos pelo grande intensivista Jean Louis Vincent, como uma espécie de checklist dos tópicos chave no manejo do paciente grave em ambiente de terapia intensiva. Mais tarde, Vincent e Hatton atualizaram o mnemônico com acréscimo de novos tópicos, FAST HUGS BID. Diversas versões foram sendo criadas, inclusive, com foco no paciente cirúrgico, que será descrita neste texto.

Como prescrever um paciente em UTI? FAST HUGS BID
Segue o texto com cada tópico, seguido de algumas perguntas que você deve se fazer ao avaliar cada paciente.

Feeding / Alimentação
Como podemos otimizar a alimentação deste paciente? Há boa aceitação da dieta via oral? Há necessidade de suplementação? A dieta está ajustada para a necessidade deste paciente? Há necessidade de dieta por sonda nasoentérica devido à inapetência? Se em dieta zero, há possibilidade de retornar a alimentação? Se em jejum por pouco tempo, está com glicose de manutenção? Há necessidade de nutrição parenteral?

Analgesia / Analgesia
O paciente está sentindo dor? A Analgesia multimodal está sendo empregada? Métodos objetivos ou subjetivos estão sendo usados na avaliação da dor? Cuidado com os Antiinflamatórios Não Esteroidais e procure usar opioides só quando necessário.

Sedation / Sedação
O paciente está confortável ou com RASS -5? Há possibilidade de reduzir a sedação? Procure minimizar o uso de benzodiazepínicos! Se for um paciente idoso, procure medidas para minimizar o risco de dellirium.

Thromboprophylaxis / Tromboprofilaxia
Todo paciente cirúrgico merece profilaxia para tromboembolia venosa, seja ela mecânica ou medicamentosa! Se em uso de heparina, há necessidade de ajuste de acordo com a função renal? Opções alternativas estão sendo usadas para pacientes com alto risco de sangramento? É possível estimular deambulação precoce?

Head-up / Cabeceira e conforto
A cabeceira da cama está elevada? Esse paciente possui risco para broncoaspiração? Se paciente restrito ao leito, ele está sendo mudado de posição a cada duas horas? Há possibilidade de colchão casca de ovo?

Ulcer prophylaxis / Proteção Gástrica
Esse paciente precisa de profilaxia de Úlcera Gástrica de Estresse? Quando podemos suspender a profilaxia? Há sensação de empachamento pós-prandial? Se histórico de úlceras ativas recentemente, há necessidade de Inibidor de Bomba de Prótons em dose plena?

Glycemic control / Controle Glicêmico
O HGT está sendo realizado nos horários corretos? A glicemia deste paciente está no nível adequado (entre 80 e 180 mg/dl)? É diabético com uso de medicações antes da internação? Há necessidade de iniciar Insulina Regular ou NPH de horário?

Spontaneous breathing trial / Teste de Respiração Espontânea
Se intubado, já está na hora do teste de respiração espontânea? De acordo com a clínica e com a gasometria, há necessidade de suplementação de O2? Se sim, em cateter nasal, máscara ou suporte de ventilação com VNI? Se em oxigenioterapia, há possibilidade de desmame?

Bowel movement / Trânsito Intestinal
Há quantos dias este paciente está sem evacuar? Se em pós-operatório, lembre-se do íleo adinâmico! Há necessidade de dieta laxativa ou de soluções medicamentosas? A deambulação está sendo estimulada? Se já está com algumas medidas, elas estão surtindo efeito ou precisam de otimizações?

Indwelling catheter / Cateteres e Sondas
A Sonda Vesical de Demora ainda é necessária? Acessos Venosos Centrais já podem ser removidos? Medicações podem ser convertidas de endovenosas para via oral, visando remoção de acessos periféricos?

Drug de-escalonation / Descalonamento de antibióticos
Foram coletadas culturas antes do início da antibioticoterapia? De acordo com o antibiograma, há possibilidade de descalonamento? Até que dia será necessário manter o antibiótico?

FAST HUGS BID – O que perguntar e o que esperar?
1. Feeding (Alimentação)
Como podemos otimizar a alimentação deste paciente?
→ Garantindo aporte nutricional adequado para evitar desnutrição e perda muscular.

A dieta está sendo bem tolerada?
→ Se não, considerar adaptação da via de alimentação (oral, enteral ou parenteral).

Se em jejum, há necessidade de glicose de manutenção?
→ Sim, especialmente em períodos prolongados, para evitar hipoglicemia e catabolismo excessivo.

2. Analgesia (Analgesia)
O paciente está sentindo dor?
→ Avaliar com escalas objetivas (EVA, CPOT) e ajustar analgesia conforme necessário.

Estamos usando analgesia multimodal?
→ Sim, preferindo AINEs e paracetamol, e reservando opioides apenas quando indispensáveis.

A analgesia está adequada ao quadro clínico?
→ Sim, sem sedação excessiva ou controle insuficiente da dor.

3. Sedation (Sedação)
O paciente está confortável sem estar excessivamente sedado?
→ Monitorar com escala RASS e ajustar para evitar sedação profunda desnecessária.

Podemos reduzir ou interromper a sedação?
→ Sempre que possível, minimizando benzodiazepínicos e prevenindo delírio.

A sedação está impedindo o desmame ventilatório?
→ Ajustar para evitar dependência prolongada de ventilação mecânica.

4. Thromboprophylaxis (Tromboprofilaxia)
Esse paciente precisa de profilaxia para tromboembolismo venoso?
→ Sim, seja farmacológica (heparina/enoxaparina) ou mecânica (meias compressivas, compressão pneumática).

Há necessidade de ajuste conforme função renal?
→ Sim, especialmente para anticoagulantes, monitorando risco de sangramento.

O paciente pode deambular precocemente?
→ Se possível, é a melhor medida para prevenção de trombose.

5. Head-up (Cabeceira elevada e conforto)
A cabeceira está elevada a pelo menos 30-45°?
→ Sim, para reduzir risco de broncoaspiração e pneumonia associada à ventilação mecânica.

O paciente restrito ao leito está mudando de posição regularmente?
→ Sim, a cada 2 horas, para prevenir úlceras por pressão.

Existe necessidade de colchão casca de ovo?
→ Sim, para pacientes de alto risco de lesões por pressão.

6. Ulcer prophylaxis (Proteção gástrica)
Esse paciente tem indicação de profilaxia para úlcera de estresse?
→ Sim, se crítico ou sob ventilação mecânica, com IBP ou bloqueadores H2.

Podemos suspender a profilaxia?
→ Sim, assim que o risco diminuir para evitar uso prolongado desnecessário.

O paciente apresenta empachamento ou sinais de desconforto gástrico?
→ Ajustar a dieta e, se necessário, rever a medicação.

7. Glycemic control (Controle glicêmico)
A glicemia do paciente está controlada?
→ Manter entre 80-180 mg/dL para reduzir complicações metabólicas.

O HGT está sendo realizado corretamente?
→ Sim, em horários adequados, para ajustes na insulinoterapia se necessário.

O paciente é diabético? Precisa de insulinoterapia específica?
→ Ajustar conforme histórico e necessidade durante a internação.

8. Spontaneous breathing trial (Teste de respiração espontânea)
Esse paciente pode ser testado para respiração espontânea?
→ Sim, se estável e atendendo critérios gasométricos e clínicos.

Qual a melhor estratégia para reduzir suporte ventilatório?
→ Avaliar possibilidade de VNI ou oxigênio por cateter nasal/máscara.

Se em oxigenoterapia, podemos reduzir o suporte?
→ Sim, progressivamente, conforme tolerância do paciente.

9. Bowel movement (Trânsito intestinal)
O paciente evacuou recentemente?
→ Monitorar, especialmente no pós-operatório, para evitar íleo adinâmico.

Há necessidade de dieta laxativa ou medicações?
→ Se constipado, considerar ajustes dietéticos e farmacológicos.

A deambulação está sendo estimulada?
→ Sim, pois ajuda no peristaltismo e prevenção da constipação.

10. Indwelling catheter (Cateteres e sondas)
O paciente ainda precisa da sonda vesical?
→ Se não houver necessidade absoluta, remover para prevenir infecções.

Os acessos venosos podem ser retirados ou convertidos para via oral?
→ Sempre que possível, reduzir dispositivos invasivos.

Há sinais de infecção associada a cateteres?
→ Monitorar para remoção ou troca em caso de necessidade.

11. Drug de-escalonation (Descalonamento de antibióticos)
O paciente ainda precisa do antibiótico?
→ Se possível, reavaliar para suspensão ou troca conforme cultura e antibiograma.

Foram coletadas culturas antes do início da terapia antibiótica?
→ Fundamental para direcionar o tratamento correto.

Até quando manter o esquema atual?
→ Definir um período adequado e evitar uso prolongado desnecessário.

    let currentStep = 0;
    const steps = document.querySelectorAll('.step');
    const beds = {}; // Armazena dados de cada leito

    // Função para alternar entre leitos
    function changeBed(bedNumber) {
        const bedData = beds[bedNumber] || {};
        document.getElementById('patientId').value = bedData.patientId || '';
        document.getElementById('admissionDate').value = bedData.admissionDate || '';
        document.getElementById('dischargeDate').value = bedData.dischargeDate || '';
        // Preencher outras informações do leito, se necessário
    }

    // Função para gerar o relatório
    function gerarRelatorio() {
        let score = 0;
        let relatorio = "📋 FAST HUGS BID Report\n\n";
        relatorio += `Patient ID: ${document.getElementById('patientId').value}\n`;
        relatorio += `Admission Date: ${document.getElementById('admissionDate').value}\n`;
        relatorio += `Possible Discharge Date: ${document.getElementById('dischargeDate').value}\n\n`;

        $('input[type=radio]:checked').each(function() {
            let nome = $(this).attr('name').toUpperCase();
            let status = $(this).val() == "1" ? "Adequate" : "Needs Adjustment";
            let obs = $(this).closest('.card').find('.obs').val().trim();
            score += parseInt($(this).val());
            if (obs) relatorio += `🔹 ${nome}: ${status}\n📝 ${obs}\n\n`;
        });

        // Mensagem de resultado
        let message = '';
        if (score >= 0 && score <= 5) message = 'Manutenção na unidade.';
        else if (score >= 6 && score <= 8) message = 'Avaliar alta.';
        else if (score >= 9 && score <= 11) message = 'Alta nas próximas 24h.';
        document.getElementById('resultMessage').innerText = message;

        $('#resultado').text(score);
        $('#relatorioFinal').val(relatorio);
    }

    // Função para enviar o relatório
    function enviarRelatorio() {
        const relatorio = document.getElementById('relatorioFinal').value;
        // Lógica para enviar o relatório por e-mail ou WhatsApp
        alert('Report sent!');
    }

    // Funções de navegação
    function nextStep(step) {
        steps[currentStep].classList.remove('active');
        steps[step].classList.add('active');
        currentStep = step;
    }

    function prevStep(step) {
        steps[currentStep].classList.remove('active');
        steps[step].classList.add('active');
        currentStep = step;
    }
</script>
